# 🔢 Sudoku Solver

A fully functional, interactive Sudoku Solver built with **React** and **Tailwind CSS**.

🌐 **Live Demo:** [https://suduko-solver-ashen.vercel.app/](https://suduko-solver-ashen.vercel.app/)

## Features
- ✅ Auto-solve any valid Sudoku puzzle using a backtracking algorithm
- 💡 Step-by-step hint system
- ↩️ Undo history
- ⚠️ Real-time conflict detection and highlighting
- 🌙 Light / Dark theme toggle with smooth transitions
- ✨ Animated floating number panels on both sides
- 📱 Responsive design

## Tech Stack
- **React** (functional components + hooks)
- **Tailwind CSS v4**
- **Vite**
